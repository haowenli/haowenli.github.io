#linux操作系统

##隐藏主机名：export PS1=’[\u \W]$ ‘

##linux操作（基本）
w #观察登录的成员及在做什么
who # 观察登录成员
who -m # 查看自己登录时间
last #观察登录过的账户
whoami # 查看自己登录，只显示用户名
clear #清屏
whatis 操作符 #帮助提示
whereis 操作符 #路径
操作符 –help #帮助文档
man python #帮助手册
URL ：Universal Resource Locator - 统一资源定位符
URI ： Universal Resource Identifier -统一资源标识符
wget url -O 名字 下载文件并重命名
info 操作符 #帮助文档
history # 查看历史输入的命令
history -c # 清楚输入命令的历史记录
！数字 # 执行数字行命令
！！ # 执行上一条命令
exit # 退出
logout #退出
shutdown #关机
shutdown -c #取消关机
shutdown -h now #立即关机
shutdown -h +10 # 10分钟后关机，定时
shutdown -r -h +10 # 10分钟后重启
shutdown -r -h 18:00 # 安排时间重启
reboot #重启
init 0 #关机
init 6 #重启
Linux系统有7个初始化级别
0 - 关机
1 - 单用户模式
2 - 多用户模式
3 - 多用户网络模式
4 - 保留
5 - 图形化
6 - 重启
SSH/TCP - 22

##账户相关命令
useradd 用户名 #创建用户
passwd 用户名 #设置用户口令
su 用户名 #切换用户
userdel 用户名 #删除用户

操作相关命令

pwd 查看当前工作目录
ls 查看当前目录下的文件和文件夹
ls -a 查看当前目录下所有的文件和文件夹
ls -l 长格式查看当前目录下所有的文件和文件夹
mkdir 创建文件夹
-p 在需要的时候创建父文件夹
cd 切换目录
绝对路径 以根路径/作为访问的基路径
相对路径 以当前路径作为访问路径的基路径
~ 当前用户的用户主目录如果是root用户就是/root
rmdir 删除空文件夹
touch 如果文件不存在就创建空文件（如果文件存在，如果存在就修改文件的最后访问时间）
rm 删除文件
-r 递归式删除 -recursive
-f 强制删除（不提示） -force
-i 交互式删除 -interactive
cp 文件名 新文件名（可以加路径）复制文件
-R 递归式的复制（用于复制文件夹）
mv 文件名 路径 移动文件
(同级目录下移动文件相当于重命名(mv 文件名 新文件名))
cat 连接多个文件的内容并显示 - concatenate
-n 显示行号
cat -n 文件名1 文件名2 将两个文件拼接显示
head -行数 文件名 显示指定行数的文件（从头开始）
tail -行数 文件名 显示指定行数的文件（从尾开始）
命令1|命令2 命令1的输出作为命令2的输入
head -行数 文件名 | cat -n (输出行号和前多少行数)
iconv -f gb2312 -t utf-8 文件名 文件转码显示，不改变文件
iconv -f gb2312 -t utf-8 文件名>新文件名 重定向，文件转码，改变文件，重新生成文件
more 文件名 按屏幕显示文件内容
less 文件名 按屏幕查看文件内容
rev 文件名 按行翻转显示文件内容
tac 文件名 倒转显示文件内容
od 文件名 以二进制（十六进制）显示文件内容
grep 搜索文件内容
cat -n hello | grep ““ 搜索文件内容 并带行号显示
正则表达式匹配 -E “\<script\s+.>” find 查找文件
find / #指定目录 find . #当前目录
-name “文件名” 根据文件名搜索
-size (+，-，数字kb，M) 根据文件大小搜索
-ctime +-数字 根据创建时间搜索(大于小于几天)
-atime 根据访问时间
-mtime根据文件修改时间
-exec 额外操作
find . -name “.html” -exec rm -f ‘{}’; #查找并删除文件

wc 统计字符数
-l 行数
-w 单词数
-c 字符数

echo (相当于输出)
> 输出重定向(echo apple > fruit.txt) 会覆盖
> > 输出重定向(echo apple >> fruit.txt) 会向下追加

< 输入重定向
2> 错误输出重定向
2> >错误输出追加重定向

文件简易操作
uniq 去重(只能相邻去重)
-c (统计重复次数)
sort 排序(正序)
-r (倒序)

cut 裁剪
split
paste粘贴
file 查看文件类型

ln 建立新链接(硬链接)
ln -s 建立软链接
软链接和硬链接的区别：软链接是创建快捷方式不添加引用计数，硬链接是增加新变量增加引用计数。

sort 文件名| uniq (全部去重了，不只是相邻去重) |cat -n (打印显示行号)(日志中可能会使用)
sort 文件名| uniq -c (全部去重了，不只是相邻去重)(统计重复次数)
tr ‘内容1’ ‘内容2’ 内容2替换输出的内容1 (cat -n fruit.txt |tr ‘a’ ‘@’)

chmod u+x 文件名 加执行权限
chmod g+x，o+x 文件名 同组或其他组加执行权限

##服务器安装python3.7

  1. 安装编译Python源代码的依赖项
    yum -y install zlib-devel bzip2-devel openssl-devel ncurses-devel sqlite-devel readline-devel tk-devel gdbm-devel libdb4-devel libpcap-devel xz-devel libffi-devel

  2. 安装前的配置 —> Makefile
    ./configure –prefix=/usr/local/python37 –enable-optimizations #指定路径，启用优化

  3. 构建和安装
    make && make install #构建，构建成功才安装
    make clean 清除构建

  4. 注册环境变量（临时）
    export PATH=$PATH:/usr/local/python37/bin
    Hao 2019/5/29 11:34:47
    echo $PATH

  5. 给python3做一个快捷方式（符号链接/软链接）
    ln -s /usr/local/python37/bin/python3 /usr/bin/python3

###yum总结
yum search - 搜索仓库中的软件包
yum -y install - 安装软件包
yum upgrade - 更新软件包
yum remove/erase - 删除
yum update - 更新所有软件包
yum list installed - 查看已安装的软件包
yum info - 查看详细信息

###启停服务
systemctl start nginx - 启动
systemctl restart nginx - 重启
systemctl stop nginx -停止
systemctl status nginx - 查看状态
systemctl enable nginx - 开机启动
systemctl disable nginx - 开机不启动
