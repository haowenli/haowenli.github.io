##vim -文本编辑器 (visual editor modified)
命令模式 - 按键都是对应的命令
编辑模式 - 可以编辑内容
末行模式 - 可以键入命令配置vim
dd 删除一行 ndd 删除n行
d0 删到行首
d$ 删到行尾
dw 删一个单词
u 撤销
yy 复制
p 粘贴
set nu 行号
set nonu 取消行号
set ts=4 设置空格为4

##vim配置文件
用户首页
创建.vimrc文件
set nu 设置行号
set ts=4 设置制表符为4个空格
set autoindent 设置为自动换行缩进
set expandtab 设置制表符扩展
syntax on 高亮语法
set ruler 设置标尺
